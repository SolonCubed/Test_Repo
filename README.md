# LIS 545 Repository Project
Repository Term Project for LIS545 - Data Curation

This repository includes the files from the following study on Gamblification, Youth and Digital Games

Aranda, D., Fernández-de-Castro, P.; & Sáenz-Leandro, R. (2025). Gamblification, Youth and Digital Games: A Dataset on Gaming Practices and Emotional Impacts Among Spanish Young People in Spain. [Data set]. [Figshare]. DOI: https://doi.org/10.6084/m9.figshare.30467558.v2.

In the Data Folder:
- Survey Results (csv)
- Codebook (xls)
- Survey Questions (pdf - En Español)
- Replication Data (r)

Repository Reporting Measures:
- Gamblification Data Folder Metadata (xml)
- Repository Report (pdf)
- Readme (txt)
